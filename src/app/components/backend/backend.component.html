2. Execute o projeto com Maven ou via IntelliJ.
3. Adicione o bot no seu grupo do Telegram como administrador e comece a conversar com ele.

Aprendizados:
- Criação e configuração de bots com Java.

- Tratamento de texto e busca de mensagens.

CRUD Simples em C#

C#

📝Descrição

Este projeto consiste na implementação de um CRUD (Create, Read, Update, Delete) simples nas linguagens C# e Java, utilizando aplicações de console para reforçar conceitos fundamentais de programação estruturada e manipulação de dados em listas.

O projeto foi desenvolvido com o objetivo de aprimorar meu conhecimento em lógica de programação, estruturas de controle e manipulação de coleções em ambas as linguagens. A ideia surgiu como um desafio de aprendizado, com suporte e orientação de um assistente de IA, me permitindo estruturar melhor o código e compreender boas práticas.

⚙ Como Funciona

O programa permite gerenciar uma lista de usuários, realizando as seguintes operações:

✅ Criar → Adicionar um novo usuário à lista
✅ Ler → Exibir todos os usuários cadastrados
✅ Atualizar → Modificar o nome de um usuário existente
✅ Deletar → Remover um usuário da lista

Cada operação é feita através de um menu interativo que o usuário pode navegar digitando a opção desejada.

🖥 Tecnologias Utilizadas


🔹 C# (.NET 6+) – Desenvolvido no Visual Studio 2022
🔹 Listas Dinâmicas – List<string> em C#
🔹 Entrada e saída de dados via console

📌 O que Aprendi com Este Projeto


💡 Refinei o entendimento sobre CRUD e sua aplicação prática, não apenas em banco de dados, mas também na manipulação de coleções em memória.
💡 Pratiquei estruturas de controle como loops e condicionais, fundamentais para manipulação de dados.
💡 Comparei a sintaxe e as diferenças entre C# e Java, aprofundando meu conhecimento em ambas as linguagens.
💡 Melhorei minha organização de código, estruturando melhor funções para cada operação do CRUD

🔗 Próximos Passos


➡ Implementar persistência de dados com um banco de dados SQL (SQLite, MySQL, PostgreSQL).
➡ Criar uma interface gráfica para tornar o projeto mais intuitivo.
➡ Desenvolver uma API REST para o CRUD e consumir os dados em um frontend.

1️⃣ using System; e using System.Collections.Generic;
🔹 System → Permite usar funções do C# como Console.WriteLine().
🔹 System.Collections.Generic → Necessário para usar a List<string>, que é uma lista dinâmica.
2️⃣ static List<string> usuarios = new List<string>();
🔹 Cria uma lista de strings chamada usuarios para armazenar os nomes dos usuários.
🔹 A lista é dinâmica, ou seja, podemos adicionar e remover elementos.
3️⃣ static void Main()
🔹 É o ponto de entrada do programa, onde começa a execução.
🔹 Possui um while (true) que mantém o programa rodando até o usuário escolher sair.
4️⃣ Estrutura de menu com switch-case
🔹 Exibe opções para o usuário e lê a entrada do console (Console.ReadLine()).
🔹 Compara a opção digitada e chama a função correspondente (CriarUsuario(), ListarUsuarios(), etc.).
🔹 Caso escolha 5, o programa encerra.
5️⃣ static void CriarUsuario()
🔹 Pede um nome e adiciona na lista usuarios.Add(nome).
🔹 Exibe uma mensagem confirmando.
6️⃣ static void ListarUsuarios()
🔹 Se a lista estiver vazia, exibe uma mensagem.
🔹 Caso contrário, percorre a lista com um for e exibe os usuários.
7️⃣ static void AtualizarUsuario()
🔹 Chama ListarUsuarios() para mostrar os usuários.
🔹 Pede ao usuário que informe o número do usuário a ser atualizado.
🔹 Usa int.TryParse(Console.ReadLine(), out int index) para converter a entrada em número.
🔹 Se o número for válido, substitui o nome antigo pelo novo: usuarios[index - 1] = novoNome.
8️⃣ static void DeletarUsuario()
🔹 Chama ListarUsuarios() para mostrar os usuários.
🔹 Pede ao usuário que informe o número do usuário a ser removido.
🔹 Se o número for válido, usa usuarios.RemoveAt(index - 1) para deletar o item.

Python - API REST

API REST - Python

API REST

Este projeto é uma API RESTful desenvolvida com Python e FastAPI, ideal para iniciantes que desejam aprender os fundamentos de APIs. Ele implementa operações básicas de CRUD (Create, Read, Delete) em memória, utilizando uma estrutura modular e escalável.

Funcionalidades:
- Adicionar itens: Endpoint POST /items/ para adicionar novos itens.
- Listar itens: Endpoint GET /items/ para visualizar todos os itens cadastrados.
- Deletar itens: Endpoint DELETE /items/{{ '{' }}item_id{{ '}' }} para remover itens pelo índice.
- Página inicial: Endpoint GET / com uma mensagem de boas-vindas.
Tecnologias Utilizadas:
- Linguagem: Python
- Framework: FastAPI
- IDE: Visual Studio Code
- Gerenciamento de Dependências: pip e virtualenv
Como Usar:
1. Clone o repositório e instale as dependências.
2. Inicie o servidor com uvicorn main:app --reload.
3. Acesse o Swagger UI em http://127.0.0.1:8000/docs para testar os endpoints.
Aprendizados:
- Criação de rotas RESTful usando FastAPI.
- Estruturação de projetos em Python com separação modular.
- Uso do Swagger UI para documentar e testar APIs.
