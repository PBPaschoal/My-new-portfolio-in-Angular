Java - Emprestimo Consignado

Java

Simulador Emprestimo Consignado - Programação Orientada a Objetos

Este é um projeto desenvolvido para praticar os conceitos de Programação Orientada a Objetos (POO).

Criado por mim em janeiro de 2024, este sistema representa a sexta versão de um projeto de empréstimo consignado que já desenvolvi anteriormente em Java e C#, mas sem utilizar POO.
Nesta versão, implementei POO para tornar o código mais modular e organizado. O sistema é simples, mas planejo expandi-lo futuramente, adicionando novas funcionalidades e uma interface gráfica para melhorar a experiência do usuário.
