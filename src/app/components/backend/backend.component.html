App Gerenciador de Tarefas

Java, Spring Boot, Angular - API REST

API REST

Este projeto destaca o desenvolvimento do backend usando Java com Spring Boot, aplicando o padrão API REST para gerenciar e expor endpoints de forma eficiente. A API permite a comunicação fluida entre o cliente e o servidor, oferecendo suporte a operações essenciais, como leitura, criação e atualização de dados.

O backend utiliza o banco de dados em memória H2, facilitando testes e desenvolvimento local sem a necessidade de configurações complexas. Além disso, todas as funcionalidades foram testadas com o Postman, garantindo desempenho.
