SORTEIO DE VAGAS

SORTEIO

Sorteio de Vagas em Condomínio Residencial

Este projeto foi desenvolvido para um cliente que solicitou um sistema simples e eficiente de sorteio de vagas para um condomínio. O website permite que os usuários realizem sorteios de apartamentos participantes de maneira interativa e fácil de usar.

A interface conta com uma página inicial que apresenta uma mensagem acolhedora e um botão para iniciar o sorteio, além de uma lista visível de apartamentos participantes. O sorteio é gerado de forma 'aleatória', garantindo que os resultados sejam exibidos em uma ordem dinâmica.

A navegação é intuitiva, com um design responsivo que se adapta a diferentes tamanhos de tela. O site utiliza tecnologias como HTML, CSS e JavaScript para criar uma experiência de usuário fluida e agradável.

O sistema foi projetado para ser de fácil manutenção e com código limpo, utilizando funções JavaScript para garantir a aleatoriedade do processo, sempre com foco na experiência do usuário. Além disso, o projeto conta com uma estrutura de layout leve e funcional, ideal para ser utilizado em ambientes residenciais ou corporativos.

Desenvolvido para praticar HTML5, CSS3 E JS.
Está é a terceira versão que crio para o cliente. Alterando alguns partes do código, textos e imagens.

TRANSPORTES

TRANSPORTES

Transporte e Logística Internacional

Este projeto é um website desenvolvido para uma empresa fictícia de transporte e logística, com foco em soluções internacionais de transporte terrestre, aéreo, marítimo e de cargas especiais. A plataforma foi construída utilizando HTML, CSS, JavaScript e frameworks como Bootstrap e UIkit, para garantir um design responsivo e interativo.

O site apresenta um layout moderno e dinâmico, com seções informativas sobre os serviços da empresa, incluindo transporte rodoviário, aéreo, marítimo e cabotagem, além de um carrossel de imagens e cards detalhados sobre cada tipo de serviço oferecido. A navegação é otimizada para dispositivos móveis, com um menu adaptável e animações suaves que aprimoram a experiência do usuário.

Além disso, a página "Quem Somos" destaca a missão e valores da empresa, oferecendo aos visitantes uma visão clara e profissional sobre o que a Exemplo pode oferecer no campo da logística internacional.

Desenvolvido para praticar HTML5, CSS3, JS, BOOTSTRAP E OUTROS.

Website: VETERINARIO

VETERINARIO

Dr. Guilherme Vicentin

Este projeto é um website desenvolvido para praticar habilidades em HTML5, CSS3, JavaScript e frameworks como Bootstrap. O site foi criado com foco em um design responsivo e atrativo, representando um portfólio para o Dr. Guilherme Vicentin, um médico veterinário ultrassonografista.

O website apresenta várias seções interativas, incluindo uma introdução com informações de contato, um menu de navegação intuitivo, uma seção explicativa sobre ultrassonografia, uma galeria de imagens e uma área 'Sobre Mim' que destaca os serviços e qualidades do profissional. Cada elemento do site foi projetado para criar uma experiência amigável e profissional para os visitantes.

Além de praticar conceitos básicos e avançados de desenvolvimento web, este projeto inclui animações suaves, um layout adaptado para dispositivos móveis e um rodapé personalizado, reforçando os detalhes do branding fictício.

Desenvolvido para praticar HTML5, CSS3, JS E BOOTSTRAP.
