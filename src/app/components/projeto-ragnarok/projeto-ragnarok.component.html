Website & Servidor - Ragnarok

Projeto Ragnarok Online

Website - Servidor - Client

Website desenvolvido com Angular, HTML5, CSS3, Bootstrap, JavaScript e TypeScript. Integração com PHP e banco de dados via phpMyAdmin.
Integração Sistema de pagamentos com Mercado Pago.

No backend, o projeto também envolve a configuração e manutenção de um servidor de Ragnarok Online utilizando o emulador rAthena, com criação e edição de NPCs, itens, scripts, imagens, sprites e otimizações.

Stack Utilizada:
Script NPC • Angular • TypeScript • HTML5 • CSS3 • Bootstrap • JavaScript • PHP • MySQL • rAthena • Mercado Pago API
